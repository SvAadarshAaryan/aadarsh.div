HTML basic format

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
  
  </body>
</html>

Scroll left to view the entire paragraph below.

1. <!DOCTYPE html>
This is a declaration, not a tag.
It tells the browser:
👉 “This page is written in HTML5.”
Without this, browsers might switch to “quirks mode,” which can break your layout.

2. <html> ... </html>
The root element of the page.
Everything on your website goes inside this tag.
👉 Think of it as the outer box that holds your whole web page.

3. <head> ... </head>
The information section about the page (not shown to visitors).
It usually contains:
• The <title> (page name in browser tab)
• Links to CSS files
• Meta tags (SEO, description, keywords)
• Scripts
👉 Basically, the settings and background info of your page.

4. <title> ... </title>
The title of the page.
👉 Whatever you write here appears on the browser tab.
Example: if you write <title>My Website</title>, the browser shows “My Website” at the top.

5. <body> ... </body>
This is the visible part of your webpage.
Everything inside <body> is shown to the visitor:
• Text
• Images
• Videos
• Buttons
• Links
👉 It’s the actual content box that people see.

6. Closing <head>, <body>, <html>
These are the closing tags for each section.
HTML always works in pairs: <tag> ... </tag>.
They tell the browser:
👉 “This part is finished.”

✅ So, your code basically says:
Use HTML5 (<!DOCTYPE html>)
Start the HTML page (<html>)
Set up background info (<head>) with a title
Show content (<body>)
Close everything properly
